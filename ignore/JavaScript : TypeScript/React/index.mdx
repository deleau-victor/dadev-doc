Dans ce cours vous allez apprendre à maitriser React autours de projet et exercices concrets dans le but d’en comprendre tous les détails afin que vous soyez en pleine capacitée de créer des application React complètes, interactives, optimisées, fiables. Vous apprendrez également les bonnes pratiques liées à cette technologie.

Ce que nous allons voir dans ce cours :

- Création de composants complets et réutilisables
- Les hooks de React pour gérer différentes fonctionnalités
- Développer une application multipage et utiliser react-router
- L’API de contexte de React afin de gérer des états globaux
- Gestion des états avec la librairie Redux
