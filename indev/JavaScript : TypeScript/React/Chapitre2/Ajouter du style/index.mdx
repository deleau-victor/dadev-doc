Il existe différentes manières d'ajouter du style dans React, pour cela, nous allons voir six différentes manières d'en ajouter :

-  Le style inline via la propriété JSX
-  Le CSS en React
-  Les modules CSS
-  Le framework CSS Styled Component
-  L'intégration du framework Tailwind CSS
-  Tailwind CSS avec le préprocesseur CSS SASS

Cependant, comme ce cours ne porte pas sur le CSS, je vous proposerai seulement une introduction aux différentes méthodes qui permettent d'en réaliser avec React.

Il sera ensuite à vous de déterminer quelle méthode vous correspond le mieux !

Cependant, je vous recommande fortement d'utiliser **Styled components** ou **Tailwind CSS** voir les **modules CSS** si vous êtes un puriste du CSS.
