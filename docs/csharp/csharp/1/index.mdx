C# (prononcé _C sharp_) est un langage de programmation à usage général, évolué et orienté objet dont la syntaxe est proche de celle du C++ et du Java. Il a été développé par Microsoft dans le cadre de sa plateforme .NET. Il est utilisé pour le développement d'applications Windows, Web et mobiles.

Ses caractéristiques ainsi que la richesse de son écosystème en font un langage très populaire. Le c# est notamment largement utilisé pour le développement de de logiciels d'entreprise, de jeux vidéo et d'applications web.

Quelques chiffre et faits à propos de C# en 2023 :

- 4<sup>ème</sup> langage de programmation le plus populaire selon l'index PYPL
- .NET est téléchargé plus de 1,5 millions de fois par mois
- Il y a plus de 7 millions de développeurs C# dans le monde
- C# est le langage de programmation le plus utilisé pour le développement de jeux vidéo

En 2023, C# est un langage de programmation à fêté ses 21 ans.
